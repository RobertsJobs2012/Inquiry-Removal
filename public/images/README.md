Place site image assets in this folder.

Suggested organization:

- `brand/`: logos, wordmarks, icons, favicon source files
- `team/`: founder photos, headshots, team images
- `results/`: client result screenshots, before/after proof, testimonial visuals
- `misc/`: any other site images that do not fit the folders above

Recommended file types:

- Logos: `svg`
- Photos: `webp` or optimized `jpg`
- Screenshots/results: `png` or `webp`

Examples:

- `/images/brand/state48-logo.svg`
- `/images/team/robert-headshot.webp`
- `/images/results/client-score-jump-01.png`
